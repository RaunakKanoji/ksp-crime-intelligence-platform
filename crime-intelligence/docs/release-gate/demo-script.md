# Demonstration Script

1. Open the public app URL.
2. Sign in with the approved demo account.
3. Show Overview and point out the sample-data label.
4. Open Records.
5. Search by district or category.
6. Open one FIR record from the result table.
7. Return to Records using browser back.
8. Open Crime map.
9. Select one incident from the map incident list.
10. Toggle Hotspots or Boundaries in map layers.
11. Open Analytics and show Districts or Trends.
12. Open Assistant and run: “Show theft cases in Bengaluru in the last 6 months.”
13. Review explanation, limitations, and source notes.
14. Open Reports.
15. Generate preview and export PDF or CSV.
16. If using an admin demo role, open Administration and show Role access or Users.
17. Open the user menu and sign out.

Fallback if a network request fails: use the visible retry/status state and explain that user input is preserved.
