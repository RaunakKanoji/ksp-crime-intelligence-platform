# Responsive Screenshot Set

Screenshots should be captured for the production deployment at these widths:

| Width | Required pages |
| ---: | --- |
| 1440 px | Overview, Records, Crime map, Analytics, Assistant, Reports, Administration |
| 1280 px | Overview, Records, Crime map, Reports |
| 1024 px | Records, Crime map, Assistant |
| 768 px | Navigation drawer, Records, Reports, Admin dialog |
| 390 px | Auth, Navigation drawer, Records, Assistant, Report form |
| 320 px | Auth, Records filter form, User menu, Admin dialog |

Release acceptance requires no hidden primary action, overlapping controls, off-screen dialogs, whole-page horizontal scrolling, or unreadable text. Data tables may scroll within their own containers.
